テストクエリ
該当のjsファイルに対して
node {JSファイル}
でサーバーたてた後、
http://localhost:4000/graphql
にアクセスし、各クエリと変数を代入してデータを取得する

・server_graphql.js
とりあえず入門としてhelloWorld

----QUERY --------------------------------------------------
{
  hello
}

・server_graphql2.js
与えた引数(条件)に対応するデータが取得できていることを確認

----QUERY --------------------------------------------------
query getSingleCourse($courseID: Int!) {
 course(id: $courseID) {
  title
  author
  description
  topic
  url
 }
}

----QUERY VARIABLES-----------------------------------------
{
 "courseID":1
}
------------------------------------------------------------

・配列が取得できることも以下のクエリと変数で確認

----QUERY --------------------------------------------------
query getCourses($topic: String!) {
 courses(topic: $topic) {
  title
  url
 }
}

----QUERY VARIABLES-----------------------------------------
{
 "topic":"Node.js"
}

・server_graphql3.js
Mulation(更新処理)について確認。取得されたデータがソースコードに記載したデータと異なることを確認。
Idで検索して、topicを更新。またfragmentの挙動も確認。

----QUERY --------------------------------------------------
mutation updateCourseTopic($id: Int!, $topic: String!) {
  updateCourseTopic(id: $id, topic: $topic) {
    ... courseFields
  }
}
fragment courseFields on Course {
 title
 author
 description
 topic
 url
}

----QUERY VARIABLES-----------------------------------------
{
 "topic":"Node.js"
}
------------------------------------------------------------

・server_graphql8.js
任意の数、ランダムに単語を取得する
----QUERY --------------------------------------------------
query getRandomWords($num: Int!) {
  randomWords(Num: $num) {
    ID
    Word
  }
}

----QUERY VARIABLES-----------------------------------------
{
  "num": 10
}
------------------------------------------------------------


//curlの取得コマンド
curl -X POST http://localhost:4000/graphql -H "Content-Type:application/json" -d '{ "query": "query getRandomWords{ randomWords(Part: \"verb\") {ID} }"}'

//curlの取得コマンド
curl -X POST http://localhost:4000/graphql -H "Content-Type:application/json" -d '{ "query": "query getRandomWords($Part: String!) { randomWords(Part: $Part) {ID,Word} }", "operationName": "getRandomWords", "variables": { "Part": "verb" } }'
